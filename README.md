# LOTUS Website

Static marketing site for LOTUS Painting & Maintenance Inc.

Hosted on Cloudflare Pages. Custom domain: lotuspaintingandmaintenance.com

- `index.html` — landing page
- `privacy.html` — privacy policy (route: `/privacy`)
- `terms.html` — terms of service (route: `/terms`)
- `styles.css` — shared stylesheet
- `_redirects` — Cloudflare clean-URL rules
